# Network Port Scanner

This project is a Python-based Network Port Scanner developed as part of the VOIS AICTE Internship.

## Features
- Scans open ports on a target system
- GUI-based interface using Tkinter
- User-friendly input for target and port range

## Technologies Used
- Python
- Socket Programming
- Tkinter

## How to Run
1. Install Python
2. Run the file:
   python port_scanner_gui.py
