# 🌐 Network Port Scanner

A Python-based GUI Network Port Scanner developed as part of the VOIS AICTE Internship.

---

## 📌 Project Description
This application allows users to scan a range of ports on a target system (IP address or hostname) to identify open ports. It provides a simple graphical interface for easy interaction and real-time results.

---

## 🚀 Features
- Scan ports within a user-defined range  
- Detect open ports on a target system  
- Multithreading for faster scanning  
- Progress bar to track scanning status  
- Stop button to terminate scanning anytime  
- User-friendly GUI using Tkinter  

---

## 🛠️ Technologies Used
- Python  
- Socket Programming  
- Tkinter (GUI)  
- Threading  

---

## 👥 End Users
- Students learning networking  
- Cybersecurity beginners  
- Network administrators
